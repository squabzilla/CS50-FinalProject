Game mechanics

The character-creation mechanics function very similarly to 5E, with a couple of notable exceptions around your maximum hit-points:

All races have a base-hitpoints, which is added to your hitpoints total.

This is done to buffer the starting hit-points, so that level 1 characters aren't quite so squishy. Also, classes grant a fix-amount of HP per level, rather then either rolling a dice, or using the averge die-roll of a dice.

So the formula for maximum-hitpoints is:

Base-Race-Hitpoint + Character-Level * ( Class-HP-Per-Level + Constitution-Modifier )
(Please remember to use correct BEDMAS order-of-operations when calculating this formula.)
